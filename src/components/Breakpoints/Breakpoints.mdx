# Breakpoints

Breakpoints define as larguras de tela em que o design da interface muda para acomodar diferentes tamanhos de dispositivos. Eles são fundamentais para criar um design responsivo.



## Exemplo de Aplicação

A seguir estão exemplos de como esses breakpoints podem ser utilizados em um componente de layout:

```css
/* Exemplo de CSS */
@media (min-width: 576px) {
  .container {
    padding: 1rem;
  }
}

@media (min-width: 768px) {
  .container {
    padding: 2rem;
  }
}

@media (min-width: 992px) {
  .container {
    padding: 3rem;
  }
}

@media (min-width: 1200px) {
  .container {
    padding: 4rem;
  }
}
