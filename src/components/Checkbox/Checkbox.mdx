# Checkbox
<p>
  A checkbox is a graphical user interface element that allows users to select one or more options from a set of choices. It typically appears as a small square box that can be checked (selected) or unchecked (deselected) by clicking or tapping on it. Checkboxes are commonly used in forms, settings, and interactive interfaces to enable users to make selections or indicate preferences.
</p>
