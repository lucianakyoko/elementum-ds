# Toggle
<p>
  A toggle is a user interface element that allows users to switch between two states, typically representing an on/off or enabled/disabled state. It consists of a graphical control, often resembling a small switch or button, that can be toggled between two positions to change the state of a particular feature, setting, or functionality.
</p>
